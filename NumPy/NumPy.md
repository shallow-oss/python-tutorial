# NumPy

## 1.常用方法
```python
np.cumsum(a)
# 计算数组的累计和，返回的还是数组
# input：[1, 2, 3, 4, 5]
# output：[1, 3, 6, 10, 15]
```

```python
np.cumprod(a)
# 计算数组的累积乘积，返回的还是数组
# input：[1, 2, 3, 4, 5]
# output：[1, 2, 6, 24, 120]
```

```python
np.inner(a, b)
# 计算两个数组的内积
# input：[1, 2, 3] [4, 5, 6]
# output：32 (张量积)
```

```python
a.T
# 转置数组
```

## 2.索引

[[1, 2, 3],
 [4, 5, 6],

A[:, 0]
